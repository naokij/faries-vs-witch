# 封面提示词编写指南

## 概述

本文档记录了仙子大战女巫有声绘本系列封面图片的提示词编写规范和参考模板。

## 封面风格规范

基于EP.11-13封面分析，统一封面风格如下：

- **画风**：日式Q版（Chibi）手绘插画风格，治愈系萌系
- **头身比**：2-3头身比例
- **构图**：对称式构图，角色居中
- **背景**：魔法森林/花园，清晨阳光，丁达尔光效
- **色彩**：高饱和度但柔和，暖色调为主
- **光效**：大量星形闪光和光点，营造梦幻氛围
- **标题**：圆润可爱的字体，白色描边，粉色填充，有光晕效果

## 提示词编写规范

### 1. 角色描述规则

**重要：不要直接写角色名字，用角色编号+参考图，避免AI误解！**

错误示例：
```
左边是妮妮，穿着绿色蝴蝶结吊带裙
右边是可可，穿着黑色外套
```

正确示例：
```
角色3（参考图3）站在角色1左侧，穿着绿色蝴蝶结吊带裙
角色4（参考图4）站在角色1右侧，穿着黑色外套
```

### 2. 提示词结构

```
[风格描述]

[主角描述（中央位置）]

[配角描述（围绕主角）]

[反派描述（特定位置）]

[背景描述]

[光效和氛围]

[标题文字描述]

[整体色彩描述]
```

### 3. 角色与参考图对应关系

在提示词开头必须提供角色与参考图的对应关系：

```
参考图对应关系：
角色1 = [角色名称]（参考图1）
角色2 = [角色名称]（参考图2）
角色3 = [角色名称]（参考图3）
角色4 = [角色名称]（参考图4）
```

## 构图建议

### 光与暗对比构图
- 反派角色放在画面上方或角落（幽暗区域）
- 主角和伙伴放在画面下方或中央（明亮区域）
- 避免把伙伴和反派放在同一侧，防止AI误解

### 角色位置原则
- 主角在画面中央
- 伙伴在主角两侧（保护姿态）
- 反派在画面上方或对面（威胁来源）

## 注意事项

1. **不要使用角色名字**：用角色编号+参考图，避免AI把名字理解成实体
2. **明确角色关系**：用"保护角色1"、"攻击角色1"等明确关系
3. **构图要清晰**：光与暗的对比在上下方向，避免左右方向导致误解
4. **提供参考图**：每个角色都要有对应的参考图
5. **色彩要明确**：指定暖色调和冷色调的具体颜色
